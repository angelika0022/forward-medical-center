# Forward Medical Center Redesign

Проект по разработке альтернативного адаптивного сайта для ООО «Медицинский центр "Форвард"» (г. Уфа, Республика Башкортостан).

Проект собран как фронтенд-версия сайта с акцентом на:

- более современный визуальный стиль;
- удобную мобильную навигацию;
- быстрый сценарий записи на прием;
- отдельные страницы услуг, врачей, цен и контактов.

## Технологии

- React 19
- TypeScript
- Vite
- React Router
- CSS

## Страницы

- `/` — главная страница;
- `/services` — направления и услуги;
- `/doctors` — специалисты;
- `/prices` — прайс-лист;
- `/contacts` — контакты клиники;
- `/appointment` — онлайн-запись.

## Локальный запуск

```bash
npm install
npm run dev
```

Сборка production-версии:

```bash
npm run build
```

Проверка линтером:

```bash
npm run lint
```

## Структура проекта

```text
src/
  components/   переиспользуемые компоненты
  data/         контент проекта
  pages/        страницы сайта
  index.css     общие стили
```

## Материалы проекта

В корне проекта выше папки `site` подготовлены черновики:

- `docs/technical-specification.md` — техническое задание;
- `docs/site-structure-and-content.md` — структура сайта и пользовательские сценарии;
- `docs/usability-testing-report.md` — отчет по юзабилити-тестированию;
- `docs/presentation-outline.md` — структура презентации.

## Примечание

Это редизайн фронтенд-части. Форма записи в текущей версии не подключена к реальному backend или CRM.
